# Pantomime Paradox (Balatro mod)

This repository is primarily a **download page** for the mod.  
➡ **Download the latest release here:**  
https://github.com/TraditionalDimension/PantomimeParadox/releases/latest

---

## Requirements
- Balatro (Steam)
- Steamodded / SMODS

## Installation
### Option A (recommended): Balatro Mod Manager
- Mod Manager (GitHub): https://github.com/skyline69/balatro-mod-manager
- Website: https://balatro-mod-manager.dasguney.com/about

### Option B: Manual
1. Download the archive from the latest release:
   https://github.com/TraditionalDimension/PantomimeParadox/releases/latest
2. Extract it into your Balatro mods folder (SMODS).
3. Launch the game.

---

## Notes
- The **“Main content”** toggle in Config requires a **game restart** (it controls whether the main mod content loads; Hard/Impossible remain available).
- The New Run UI toggles are designed to **co-exist** with other mods (e.g. Partner / Galdur Compat).
